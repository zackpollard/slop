# Cron Translator

Paste a cron expression and get a human-readable description, visual timeline, and list of upcoming runs.

## Features

- Real-time parsing and description of 5-field cron expressions
- Visual field breakdown (minute, hour, day-of-month, month, day-of-week)
- Next 10 runs with relative timestamps
- Timeline visualization
- Preset expressions for common schedules
- URL hash support for sharing expressions
- Cron syntax cheatsheet

## Local development

```bash
python3 -m http.server 8000
```
