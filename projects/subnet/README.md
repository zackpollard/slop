# Subnet Calculator

Enter an IP address and CIDR prefix to get a full subnet breakdown with binary visualization and address space map.

## Features

- Network/broadcast address calculation
- Netmask and wildcard mask display
- Usable host range and count
- IP class detection (A/B/C/D/E)
- Network type identification (private, public, loopback, link-local, multicast)
- Color-coded binary breakdown showing network vs host bits
- Visual address space bar chart
- CIDR slider for quick prefix adjustment
- Preset common subnets

## Local development

```bash
python3 -m http.server 8000
```
