# JSON Carpenter

Format, validate, and diff JSON with syntax highlighting, collapsible tree view, and path copying.

## Features

- **Format & Validate**: Paste JSON, get instant syntax-highlighted formatting
- **Tree View**: Collapsible tree with type-colored values and click-to-copy paths
- **Minify**: Toggle between pretty-printed and minified output
- **Diff**: Compare two JSON objects side-by-side with added/removed/changed highlighting
- Byte count and depth statistics
- Sample JSON for quick testing
- Tab key inserts spaces in the editor

## Local development

```bash
python3 -m http.server 8000
```
