slop.

A collection of small projects and experiments by Zack Pollard.

What is this?

slop is a monorepo and home for various small apps, tools, and experiments. Each project lives in its own space and gets deployed as a subdomain under slop.zackpollard.pro.

Projects span different languages and frameworks — whatever fits the problem best. This page serves as the index and directory for everything hosted here.

Projects

Roof Calculator

Satellite map-based roof measurement tool. Draw polygons over roof sections, set pitch angles, and calculate true area with waste allowance.

D&D Encounter Generator

Combat encounter generator using D&D 2024 rules. Configure party size, level, and difficulty with environment, creature type, and theme filters.

Cron Translator

Paste a cron expression, get a human-readable description, visual timeline, and list of upcoming runs.

Subnet Calculator

Enter an IP address and CIDR prefix to get full subnet breakdown with binary visualization and address space map.

JSON Carpenter

Format, validate, and diff JSON with syntax highlighting, collapsible tree view, and path copying.
