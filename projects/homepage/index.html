slop.

A collection of small projects and experiments by Zack Pollard.

What is this?

slop is a monorepo and home for various small apps, tools, and experiments. Each project lives in its own space and gets deployed as a subdomain under slop.zackpollard.pro.

Projects span different languages and frameworks — whatever fits the problem best. This page serves as the index and directory for everything hosted here.

Satellite map-based roof measurement tool. Draw polygons over roof sections, set pitch angles, and calculate true area with waste allowance.
